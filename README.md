# Kilov

## Feature

**c.f. [ANSI Escape Sequences](https://gist.github.com/fnky/458719343aabd01cfb17a3a4f7296797)**

|機能|I/O|エスケープシーケンス|説明|
|:-:|:-:|:-:|:-|
|カーソル移動|IN|`<ESC>[999C`|カーソルを最右端に移動|
|カーソル移動|IN|`<ESC>[999B`|カーソルを最下端に移動|
|カーソル移動|IN|`<ESC>[n;mH`|カーソルを現在の位置に関係なく上端から`n`、左端から`m`の場所に移動させる|
|カーソル移動|IN|`<ESC>[H`|ホーム(0, 0)に移動する|
|カーソル位置問合せ|IN|`<ESC>[6n`|カーソル位置を問い合わせて報告を待つ|
|カーソル位置報告|OUT|||
|カーソル表示変更|IN|`<ESC>[?25l`|カーソルを非表示にする|
|カーソル表示変更|IN|`<ESC>[?25h`|カーソルを表示する|
|表示色変更|IN|`<ESC>[7m`|反転、前景色と背景色を入れ替えて表示|
|表示色変更|IN|`<ESC>[39m`|文字色をデフォルトに戻す|
|指定リセット|IN|`<ESC>[0m`|それまでの指定を無効化する|
|文字編集|IN|`<ESC>[0K`|カーソルから行末までを削除する|
